## Objetivo:
Este projeto tem por objetivo a criação de 5 mini sistemas simples, com o intuito de consolidar a pratica com a linguagem Java.
Os sistemas foram criados com o propósito de praticar:
- Programação Orientada a Objetos (POO)
- Persistência de dados com arquivos
- Tratamento de exceções
- Lógica de programação
- Organização em camadas

## Estrutura do projeto:
O projeto vai funcionar da sequinte maneira, terao arquivos com o nome do exercicio proposto e o tema, o objetivo é tentar resolver sem utilzar frameworks, e sim com o java e logica pura para ser estimulado a pesquisar e resolver as problmeaticas

## Temas
Abaixo a lista dos 5 temas que serão codados e resolvidos:
- Sistema de cadastro de Pet; ✅
- Sistema de Controle de Filmes.⏳
- Sistema de Controle de Tarefas;⏳
- Sistema de Reserva de Sala;⏳
- Sistema de Cadastro de usuarios;⏳
